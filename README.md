# YamiMap
## Basic maps usage

```
var map:YAMIMap = new YAMIMap(200, 200, this);
var provider:GoogleMapsProvider = new GoogleMapsProvider(GoogleMapsProvider.MAPTYPE_ROADMAP, 2, 'ru-RU');
map.addLayerByProvider(provider);
```

<li>
